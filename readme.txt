Для запуска приложения требуется Python 3.8 - 3.12. Библиотеки установятся автоматически.

Для запуска приложения на linux запутстите скрипт start_on_linux.sh
На ОС Windows - start_on_windows.bat

Далее необходимо открыть страницу 127.0.0.1:5000 в браузере.

Если нужно дообучить модель:
	Дополните базу data.xlsx в папке data
	Если ни разу приложение не запускалось, запустите start_on_windows.bat или start_on_linux.sh
	Нужно запустить комманду "python train_model.py"
	и далее, если приложение запущенно, нужно перезапустить приложение
