Для запуска приложения требуется Python 3.8 - 3.12. Требуется операционная система Windows. Библиотеки установятся автоматически.

Для запуска приложения запускаем start_on_windows.bat 
Далее необходимо открыть страницу 127.0.0.1:5000 в браузере.

Если нужно дообучить модель:
	Дополните базу data.xlsx в папке data
	Нужно запустить комманду "python train_model.py"
	и далее, если приложение запущенно, нужно перезапустить приложение
